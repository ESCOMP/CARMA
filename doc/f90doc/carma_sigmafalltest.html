This code is to demonstrate the CARMA sedimentation routines using a constant fall velocity. Sigma coordinates are used by the test.

Upon execution, a text file (carma_sigmafalltest.txt) is generated. The text file can be read with the IDL procedure read_sigmafalltest.pro.
