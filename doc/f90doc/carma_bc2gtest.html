This code is to demonstrate the CARMA coagulation routines in a test case utilizing two groups, one group that has two components (OC, BC/OC).

Upon execution, a text file (carma_bc2gtest.txt) is generated. The text file can be read with the IDL procedure read_bc2gtest.pro.
