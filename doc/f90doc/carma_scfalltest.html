This code is to demonstrate the CARMA sedimentation routines using a constant fall velocity. Cartesian coordinates are used in the vetical to represent a hybrid grid.

Upon execution, a text file (carma_scfalltest.txt) is generated. The text file can be read with the IDL procedure read_sigmafalltest.pro.
