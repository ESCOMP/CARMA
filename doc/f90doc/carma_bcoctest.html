This code is to demonstrate the CARMA coagulation routines in a test case utilizing three groups, one group that has two components (BC, OC, OC/BC).

Upon execution, a text file (carma_bcoctest.txt) is generated. The text file can be read with the IDL procedure read_bcoctest.pro.
