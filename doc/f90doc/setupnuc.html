This routine evaluates derived mapping arrays and calculates the critical supersaturation used to nucleate dry particles (CN) to droplets.

This routine requires that array is defined. (i.e., setupgkern.f must be called before this)

NOTE: Most of the code from this routine has been moced to CARMA_InitializeGrowth because it does not rely upon the model's state and thus can be called one during CARMA_Initialize rather than being called every timestep if left in this routine.
