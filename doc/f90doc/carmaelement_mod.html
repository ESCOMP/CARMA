end subroutine CARMAELEMENT_Create
Defines a gas used by CARMA for nucleation and growth of cloud and aerosol particles.

NOTE: The element density can be specifeid per bin using rhobin; however, if only the bulk density is provided (rho) then the same value will be used for all bins. The bulk density allows for backward compatability and ease of configuration. If rhobin is provided, then rho is ignored.

Author: Chuck Bardeen

Version: March-2010

See also: module CARMA_AddGas, module CARMAELEMENT_Destroy
